# Text Area

A text area component allows users to input multi-line text. It is commonly used in forms and web pages for entering larger amounts of text.

## Default

<code-bubble>

```html
<ch-text-area></ch-text-area>
```

</code-bubble>

## Label

To add a label, use the `label` attribute within the `<ch-text-area>` element.

<code-bubble>

```html
<ch-text-area label="Label"></ch-text-area>
```

</code-bubble>

## Help text

To add a descriptive help text, use the `help-text` attribute within the `<ch-text-area>` element.

<code-bubble>

```html
<ch-text-area help-text="help text here"></ch-text-area>
```

</code-bubble>

## Rows

To change the number of rows, use the `rows` attribute within the `<ch-text-area>` element.

<code-bubble>

```html
<ch-text-area rows="2"></ch-text-area>
```

</code-bubble>

## Placeholder

Use the `placeholder` attribute within the `<ch-text-area>` element to add a placeholder.

<code-bubble>

```html
<ch-text-area placeholder="Type something"></ch-text-area>
```

</code-bubble>

## Disabled

To disable the textarea, add the `disabled` attribute within the `<ch-text-area>` element.

<code-bubble>

```html
<ch-text-area disabled></ch-text-area>
```

</code-bubble>
